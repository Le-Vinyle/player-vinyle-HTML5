All the config settings :

  • enable
    You can set if to false to defer the initialization and load the player when you want.
          enable: true
  • mode
    You have the choice betwwen the 'automatic' and the 'manual' mode. The last one requires the user drags the arm on the disc to play it.
          mode: 'automatic'
  • debug
    Enable the debug mode by logging all the informations and events in the console.
          debug: false
  • logMethodNames
    As the player switches on/off the console methods, you can list the method which will be overriden.
          logMethodNames: ["log", "debug", "warn", "info"]
  • paths
    You can set your own directories.
          paths: { 
        audio: 'audio/',
        music: 'music/',
        playlists: 'playlists/',
        themes: ''
      }
  • ids
    You can set your own ids for the element created by the player.
          ids: {
        player: 'player',
        remote: 'remote',
        infos: 'infos',
        cover: 'cover',
        audioplayer: 'turntable-player',
        nextbutton: 'next',
        powerbutton: 'power',
        powerbuttonon: 'power-on',
        powerbuttonoff: 'power-off',
      }
  • rpm
    The round per minute of the disc. WARNING : this option will be modified soon since we will add several disc formats (12'', 10'' et 8'')
          rpm: 45
  • animateDelay
    The duration of the arm animations at the beginning and the end, in automatic mode.
          animateDelay: 2000
  • autoStop
    Duration in ms when the turntable auto-shutdowns when it turns with no track, in manual mode.
          autoStop: 600000
  • endTransitionDuration
    Duration in ms of the repetition of the end transition, in manual mode.
          endTransitionDuration: 0
  • buttonLabels
    Customize the labels of the buttons.
          buttonLabels: {
        powerON: 'I',
        powerOFF: 'O',
        next: '⟳'
      }
  • easing
    You can set different easing effect for the disc rotation.
          easing: { 
        start: '<',
        pause: 'cubic-bezier(.81, .79, .57, 1.01)',
        stop: 'cubic-bezier(.81, .79, .57, 1.01)'
      }
  • forceDateInUri
    Force the request to retrieve an updated playlist.
          forceDateInUri: true
  • playlistLocation
    Name of the playlist file.
          playlistLocation: 'playlist.json'
  • infos
    Choose the informations to display in the 'infos' panel : duration, current, timer, position.
          infos: ["duration", "timer"]
  • panels
    Choose the panels to display.
          panels: { // The panels to display
        'cover': true, 
        'infos': false, 
        'playlist': false
      }
  • useTransitions
    Enable the audio transitions
          useTransitions: true
  • useCssAnimations
    Enable the CSS animations or use SVG with Raphael.js.
          useCssAnimations: true
  • useShadow
    Enable the background drop shadow.
          useShadow: true
  • theme
    Choose your theme among the given ones or the ones your provided.
          theme: 'wood'
  • themes
    You can provided new themes to the list of themes set by default. Look at the code to create a new one.
          themes: {
        'mytheme': {
          cssClass: 'my-theme-class my-other-needed-theme-class',
          ...
        }
      }
  • transitions
    The audio transitions
          transitions: {
        start: {
          loop: false,
          src: {
            mp3: 'start.mp3',
            ogg: 'start.ogg'
          }
        },
        ...
      }
